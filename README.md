# jspong
Created with [p5.js](https://p5js.org/).

Played on a cylinder.

Pass the ball by your opponent five times to win.

# Controls

### Player 1 Controls
`up arrow` and `down arrow` - Move 

### Player 2 Controls
`A` and `Z` - Move 

### Global Controls
`Spacebar` - Pause
`Enter` - Re-serve ball
`H` - Show/hide HUD
